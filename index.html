Web Watcher

Purpose

Monitor webpages for changes and notify users by email

Features

Scalable

Web Watcher is a microservice application following the API gateway design pattern, it consists of 7 JavaScript/Python services and 1 Go Cronjob.

Secure

Clients use JWT cookies following the refresh_token and access_token strategy.

Data at rest is encrypted using Bcrypt.

Free

Web Watcher is a multi cloud solution using free services from IBM Cloud, AWS, Docker Hub and Netlify. Some notable services are IBM Kubernetes Service, Docker Hub, API Gateway, Cloud Object Storage and CouchDB.

Usable

React with IBM Carbon Design on the frontend with mobile sizing in mind.

Nitrus

Purpose

Features

Integrated

Nitrus uses Spotify API to search tracks with your preferences, synchronize playback for the entire room and stop playback on exit.

Realtime

Nitrus uses socket.io to keep votes, tracks and active users synched across clients.
